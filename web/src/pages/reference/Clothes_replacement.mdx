### Replacing models (DFF)
Starting with version **1.6.0 r22987**, CJ's clothing models can be replaced.
```lua
-- Load your custom .dff file that will
-- replace one of the game's DFFs for CJ clothing
local dff = engineLoadDFF("custom_afrobeard.dff")
engineReplaceModel(dff, 30001)
```

### Adding new custom clothing
Starting from version **1.6.0 r23124**, it is possible to add new CJ clothes using the functions [engineAddClothingModel](/reference/engineAddClothingModel), [engineAddClothingTXD](/reference/engineAddClothingTXD), and [addClothingModel](/reference/addClothingModel).
